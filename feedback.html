Feedback

Thanks to the feedback form we can continuously improve the web application and quickly be informed of any errors that were not discovered during development and testing phases.

Here you can find more information about the whole process of sending and processing of a feedback, incl. what data is being stored, what are your rights etc.

Feedback Data

The data you provide is stored in an encrypted form on the same server from which the feedback originated and no third-party service is included in the whole process.

The feedback data consists of:

  • E-mail address
  • Rating
  • Description

As mentioned above, this data is all encrypted, therefore nobody can read it unless they obtain the private key, which is securely stored outside of the web server and is known to and accessible only by the application author (that is the person mentioned in the chapter Contacts).

User data

Additionally to the data above, we store also so called metadata about the feedback, which consists of:

  • Hashed IP address - this way nobody, even the application's author, can retrieve the original IP address back from the stored value (Hash function @Wikipedia)
  • Timestamp of feedback submission
  • Earliest timestamp of next feedback submission

This metadata is just to prevent from misuse of the feedback form by bots/hackers/…

Privacy

The only personally identifiable information being collected is the optional e-mail address, which is used only for an e-mail response to that particular feedback. This e-mail address is not stored anywhere else and is not shared with anyone.

We strongly recommend not to input your personally identifiable information (name, address, phone, date of birth, social security number, credit card number etc.) in the description, as it is not needed anyway. Such feedback may even be deleted by the author to keep your information safe.

Rights

No matter where you are, you have the right to obtain the data we have collected about you or request permanent deletion of such data.

Kindly be informed that if you did not provide the e-mail address in your feedback, it might not be possible to identify your data completely.

Please refer to the chapter Contact for ways to send such request.
