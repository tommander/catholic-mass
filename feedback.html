Feedback

Anyone can optionally send a feedback to the Order of Mass web application.

This way we can continuously improve the site and quickly be informed of any errors that were not discovered during development and testing phases.

Here you can find more information about the whole process of sending and processing of a feedback, incl. what data is being stored, what are your rights etc.

Basic info

Here I would like to ask you - please be constructive about the feedback. If you noticed an error, please describe, what have you done on the page before the error appeared, what does the error say, if it happens continuously or it happened only once. If you notice a mistake in the text, please include the translation and on which page (mass, rosary, bible) etc. Simply stating it doesn't work, something happenned or it's ugly is for sure a feedback, but won't help to fix it for you and others.

Just to make sure:

  • It is not in any way obligatory that a user sends a feedback
  • Sending a feedback does not unlock any premium functionality or access to the application
  • Sending a feedback does not give a user an enforceable right to receive any kind of prize or payment

Data

The data you provide is stored in an encrypted form on the same server from which the feedback originated, i.e. no third-party service is included in the whole process.

Example of (decrypted) data submitted from a user:

{
    "email": "email@address.com",
    "rating": 1,
    "description": "It does not work",
    "metadata": {
        "ip": "abcdef1234567890",
        "useragent": "",
        "timestamp": "",
    }
}

This data is all encrypted, therefore nobody can read it unless they obtain the private key, which is securely stored outside of the web server and is known to and accessible only by the application author (see chapter Contacts).

The IP address of the user is hashed and therefore cannot be read even by the application author. This is used only for the purpose of stopping from batch feedback submission e.g. by bots. Internet is a wild place.

Privacy

The only personally identifiable information being collected is the optional e-mail address, which is used only for an e-mail response to that particular feedback. This e-mail address is not stored anywhere else and is not shared with anyone.

We strongly recommend not to input your personally identifiable information (name, address, phone, date of birth, social security number, credit card number etc.) in the description, as it is not needed.

Rights

No matter where you are, you have the right to obtain the data we have collected about you or request permanent deletion of such data.

Kindly be informed that if you did not provide the e-mail address in your feedback, it might not be possible to identify your data completely.

Please refer to the chapter Contact for ways to send such request.
